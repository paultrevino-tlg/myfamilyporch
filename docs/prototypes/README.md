# Visual prototypes

Static, self-contained HTML mockups built during design. Open in a browser.
They are the source of truth for layout, tone, copy, and interaction intent —
build the real components to match these, then delete or keep as reference.

- **storyteller-flow.html** — the elder's 7-screen session, with an EN/ES
  language toggle. Notification → Welcome → Question (cloned voice) → Your turn
  → AI follow-up → Your turn → Done. Large targets, single-tap, voice-first.
- **family-admin.html** — the family/admin dashboard. Overview (with the three
  signals — there's a PROTOTYPE toggle to see the "doing well" vs "quieter
  lately" states), Stories review, Topics steering, Schedule, The Book, Settings.

These map to TODO Phases 2 (storyteller) and 5–7 (admin, signals, book).
